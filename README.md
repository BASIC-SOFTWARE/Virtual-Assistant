# Virtual Assistant
This is a virtual assistant created by Bekhruz Niyazov for 3rd Code Jam hosted by Tech With Tim.
# Requirements
You need to have Python 3.x installed
# Usage
### Windows
```sh
python main.py
```
### macOS and Linux
```sh
python3 main.py
```
or
```sh
chmod +x main.py
./main.py
```

I hope you will enjoy using this virtual assistant!
